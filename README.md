# Moonlight

Lightweight kernel manager for Android.

## Install

Required Gradle 8.13 and JDK 17.

- **Building from source:**
```
# windows
gradle w
gradlew.bat assembleRelease 

# linux
gradle w
chmod +x gradlew
./gradlew assembleRelease 
```

- [**Install**](https://git.gay/nguyencaoantuong/Moonlight/releases)
